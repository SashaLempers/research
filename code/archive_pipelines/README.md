# Archive — Pipelines SRMT v11–v15

Ce dossier regroupe les versions intermédiaires du pipeline Macaulay2 SRMT,
de v11 à v15, conservées pour la traçabilité de la progression de la recherche.

Ces scripts ne sont **pas cités** dans les articles en cours. La version de
référence actuelle est v19, dont les scripts se trouvent dans le dépôt
[`srmt-reproducibility`](https://github.com/SashaLempers/srmt-reproducibility).
